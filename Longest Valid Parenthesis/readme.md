## Problem Decsription

Given a string containing just the characters '(' and ')', find the length of the longest valid (well-formed) parentheses substring. 

## Input 1: 
    s = "(()"
## Output 1: 
    2
## Explanation: 
    The longest valid parentheses substring is "()".

## Input 2: 
    s = ")()())"
## Output 2: 
    4
## Explanation:  
    The longest valid parentheses substring is "()()".

## Input: 
    s = ""
## Output: 
    0
