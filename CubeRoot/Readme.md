# Problem Description

Given a number n, find the cube root of n.

Notice that the solution set must not contain duplicate triplets.

# Input 1: 
    n = 3
# Output 1: 
    Cubic Root is 1.442250

# Input 2: 
    n = 8
# Output 2:
    Cubic Root is 2.000000

# Input 3: 
    n = 125
# Output 3:
    Cubic root is 5.000000

## Given precision e = 0.0000001
