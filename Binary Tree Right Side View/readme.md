# Problem Description

Given the root of a binary tree, imagine yourself standing on the right side of it, 
return the values of the nodes you can see ordered from top to bottom.

# Input 1: 
    root = [1,2,3,null,5,null,4]
# Output 1: 
    [1,3,4]

# Input 2: 
    root = [1,null,3]
# Output 2: 
    [1,3]

# Input 3:
    root = []
# Output 3: 
    []
